Heart Disease Prediction Using Machine Learning

Department of Information Technology | MVGR College of Engineering

Introduction

Heart disease is one of the leading causes of mortality worldwide. This project aims to predict heart disease risk using Machine Learning models trained on UCI benchmark dataset.

Project Abstract

Developed a predictive model using Random Forest as primary algorithm. Compared with Logistic Regression, SVM and Decision Tree. Random Forest achieved highest accuracy.
Problem Statement
Early detection of heart disease remains challenging. Current systems lack scalable and accurate predictive models.
Present System
• Medical History & Risk Assessment • Physical Examination • Lab Tests • ECG Analysis
Proposed System
Machine Learning based prediction using:
  • Random Forest
  • Logistic Regression
  • SVM
  • Decision Tree
Modules Implemented
  • Data Preprocessing
  • Feature Selection (SelectKBest)
  • Model Training
  • Model Evaluation
  • Prediction Generation
  • Result Presentation
Software & Hardware Requirements
Software: Python, Google Colab, NumPy, Pandas, Sklearn, Matplotlib, Seaborn

Hardware: Intel i3, 4GB RAM, 5GB HDD
Outputs & Performance Metrics
• Correlation Heatmap • Data Cleaning Summary • Model Accuracy Comparison • Test Case Evaluations
Conclusion
Random Forest showed highest accuracy among tested models. Future work includes deploying model in real-world healthcare systems.
