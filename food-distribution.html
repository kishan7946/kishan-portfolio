Food Distribution Management System

Web-Based Platform Connecting NGOs, Donors & Restaurants

Project Overview

This web-based application works as a mediator between NGOs, donors, and restaurant owners to reduce food wastage and help distribute leftover food to needy people efficiently.

System Flow

Restaurant

Upload leftover food details

Platform

Matches NGO requests

NGO

Collect & Distribute
Problem Statement
Large amount of food is wasted daily while many people suffer from hunger. There is no centralized system connecting food donors with NGOs efficiently.
Proposed Solution
A web application where:
  • Restaurants register and post available food
  • Donors can contribute
  • NGOs can request and collect food
  • Admin manages the overall system
Modules Implemented
  • User Registration & Login
  • Food Donation Upload
  • NGO Request System
  • Admin Dashboard
  • Notification & Tracking System
Technology Used
HTML, CSS, JavaScript Backend: PHP / Python (depending on implementation) Database: MySQL
Business Impact
• Reduces food wastage • Improves NGO coordination • Provides real-time donation tracking • Social impact driven system
